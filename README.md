Microsoft Coding QA Re-annotation Project
Worked on improving AI training data by creating high-quality reference responses for coding prompts. Analyzed user prompts and metadata to determine intent, category, complexity, and domain, then produced clear and structured explanations and code solutions following strict style guidelines. Ensured responses handled ambiguous or incomplete prompts using logical assumptions and reasoning while prioritizing prompt instructions over metadata. Contributed to enhancing model performance on coding Q&A tasks through accurate annotations, concise explanations, and well-structured solutions across multiple programming domains.
